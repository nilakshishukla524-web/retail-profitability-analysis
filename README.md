# Retail Business Performance & Profitability Analysis

## Overview
Analysis of a retail transactional dataset (9,994 records) to identify
profit-draining categories, evaluate the impact of discounting on
profitability, and study regional/segment-wise performance.

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- SQL (SQLite)
- Tableau Public

## Key Steps
1. Data cleaning (removed duplicates, validated data)
2. SQL analysis — profit margin by category, sub-category, region
3. Python correlation analysis — discount vs profit relationship
4. Interactive Tableau dashboard with filters

## Key Insight
Orders with discounts above 30-40% show negative average profit,
confirming a strong negative correlation between discount and profit.
